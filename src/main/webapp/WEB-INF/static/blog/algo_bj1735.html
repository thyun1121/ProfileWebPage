분수 합

문제

https://www.acmicpc.net/problem/1735

접근법

  1. 결과 분수의 분모 = 입력값의 분모끼리의 곱
  2. 결과 분수의 분자 = 입력값의 분자*입력값의 상대 분모 후 분자끼리 더해준다.
  3. 결과 분수의 분모와 분자의 최대공약수를 구한 후 각각 최대공약수로 나눠준다.
    • 최대공약수 구하는 법은 13241번의 접근법 참고.

풀이

java

import java.util.Scanner;

public class BJ_1735 {

	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		
		// a/b + c/d
		int a = sc.nextInt();
		int b = sc.nextInt();
		int c = sc.nextInt();
		int d = sc.nextInt();
		
		int bottom = b*d;
		int top = (a*d)+(c*b);
		
		int gcd = findGcd(bottom, top);
		
		bottom/=gcd;
		top/=gcd;
		
		System.out.println(top+" "+bottom);
	}
	
	public static int findGcd(int a, int b){
		if(a<b){
			int temp;
			temp = a;
			a=b;
			b=temp;
		}
		
		while(true){
			int r = a%b;
			if(r==0)
				return b;			
			a=b;
			b=r;
		}
	}
}
