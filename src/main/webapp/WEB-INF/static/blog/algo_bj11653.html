소인수분해

문제

https://www.acmicpc.net/problem/11653

접근법

소인수분해는 어떤 수를 소수로만 분해하는 것을 의미한다. 우리가 학창시절 소인수분해 문제를 풀 때는 소수들로만 나누어 답을 구하곤 했다. 하지만 프로그래밍으로는 어떻게 해야할까? 소수들을 따로 리스트에 저장시킨 후 소수들로 나누어야 할까?
소수들로만 따로 분해할 생각을 하지 않아도 된다. 2,3으로 다 분해하고 나서는 4 또는 6으로 분해될 일이 없다. 왜냐하면, 이미 2와 3으로 다 분해했다는 것은 2와 3으로 이루어진 4,6 또한 사용되었다고 볼 수 있기 때문이다. 그러므로 우리는 2부터 주어진 수가 1로 분해될 때까지 반복문을 실행시키면 된다.

풀이

java

package baekjoon;
import java.util.Scanner;

public class BJ_11653 {
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int N = sc.nextInt();
		
		for(int i=2; N>1;){
			if(N%i==0){
				System.out.println(i);
				N/=i;
			}else
				i++;
		}
	}
}
